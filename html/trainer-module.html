parseHTML(url)

source code 

Essentially downloads a web page, from a provided URL. Its HyperText Markup is then parsed, by means of BeautifulSoup. A regex is used to clean the web page's title, and assign the result to the output file. All lexicon tags from the file are then separated, and appended to a new string, to be returned.

Parameters:
  • url (url) - the web page's uniform resource locator
Returns: string
holds the majority of words within the original text

@rtype  void

processDocuments(documents, full_word_list)

source code 

Handles the frequency distribution, saving word features, shuffling feature sets, training the Naive Bayes classifiers etc. It also separates the training sample to a subset for training, and one, reserved for testing back. Eternally, it saves the final .pickle file in the "pickled_class" directory.

Parameters:
  • documents (string) - stores the content text of the source .pickle file
  • full_word_list (list<string>) - containts all words

    @rtype void @return console output
